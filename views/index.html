FreeCodeCamp API Basejump: URL Shortener Microservice

Example usage:

https://cedar-basketball.glitch.me/new/https://www.google.com
https://cedar-basketball.glitch.me/new/http://foo.com:80

Example creation output:

{ "original_url":"http://foo.com:80", "short_url":"https://cedar-basketball.glitch.me/4" }

Usage:

https://cedar-basketball.glitch.me/3

Will redirect to:

https://www.google.com/
